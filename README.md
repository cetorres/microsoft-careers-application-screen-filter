# Microsoft careers page - Application on Screen filter

Adds a new filter for applications on Screen status to the candidate's Microsoft careers applications page.

Script can be used with Chrome extension: [User JavaScript and CSS](https://chromewebstore.google.com/detail/nbhcbdghjpllgmfilhnhkllmkecfmpld?utm_source=item-share-cb) version 3.1.2 or above.
